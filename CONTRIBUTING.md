# Contributing to softwaredevelopmenttr-official-website

:+1::tada: First off, thanks for taking the time to contribute! :tada::+1:

The following is a set of guidelines for contributing to softwaredevelopmenttr-official-website, which is hosted in the [softwaredevelopmenttr official website](https://github.com/softwaredevelopmenttr/softwaredevelopmenttr-official-website) on GitHub. These are mostly guidelines, not rules. Use your best judgment, and feel free to propose changes to this document in a pull request.
